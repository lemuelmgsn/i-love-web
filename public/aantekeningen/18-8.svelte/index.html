Svelte opzetten

Stappen bij het opzetten van een nieuw svelte project.

  1. In de terminal:
# create a new project in the current directory
npm create svelte@latest

# create a new project in my-app
npm create svelte@latest my-app
  1. Stappen volgen in de terminal (skeleton project etc..)
  2. npm install
  3. nieuwe files toevoegen zodat svelte werkt (in de volgende stappen)
  4. in de lib (library) een fetch-json.js file toevoegen. met daarin
export default async function fetchJson(url, payload = {}) {
  return await fetch(url, payload)
    .then((response) => response.json())
    .catch((error) => error)
}
  1. in de routes een +page.server.js file toevoegen met daarin:
import fetchJson from "$lib/fetch-json"

export async function load() {
  const url = 'https://fdnd.directus.app/items/person/?filter={"squad_id":3}'

  const persons = await fetchJson(url)

  return {
    persons: persons.data
  }
}
  1. in de routes een +page.js file toevoegen met daarin:

export let csr = false

  1. Om de server te runnen:

npm run dev
